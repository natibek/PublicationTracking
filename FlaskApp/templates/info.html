Info

The website takes in the BSD faculty roaster, SCOPUS csv file, and the intended tracking excel file in that order to find all of the latest publication by BSD faculty and update the tracking excel file. Once all the files with the correct columns are uploaded, the submit button with appear. Clicking it will download the updated tracking file.

The rows for publications with no correspondence address are filled with yellow and the last author in author list is used as the corresponding author.

In cases where the Authors with affiliation column form Scopus has formating other than not last-name, first-initial, affiliation; ... resulting in a different number of authors parsed from this column than in the Authors column, the row is filled with green.

If both strange cases are true, the row is filled with blue.
